## How to add a new entry

1. Check this file first — if the company+role already exists, **do not apply again**
2. Fill the form / submit the application
3. Append a row to the table below using this format:

```

```

**Status values:** `Applied` · `Responded` · `Interview` · `Offer` · `Rejected` · `Withdrawn` · `Closed`
